# README

###Program features

This project is a CMS (Content Management System) for the MEC webapp. 
The project has an admin user that is capable of:

- CRUD Section
- CRUD Employee
- CRUD Product
- CRUD Carousel

The project also has a mailer service to contact the MEC staff.

###Versions

-Ruby version '2.6.1'
-Rails version '5.2.3'

###Deployment

The project has a script to start/restart the server. Once inside the folder of the project the command to run it is:

./restart_server.sh PROD 

The project only works under the server that is administrating api.mec.com.co and with its domain. In order to change the server, the domain or both it's recommended to contact the developer.
